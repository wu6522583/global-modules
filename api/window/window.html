扩展自 $.fn.panel.defaults，用 $.fn.window.defaults 重写了 defaults。

 

依赖

  • draggable
  • resizable
  • panel

用法

1.  <div id="win" iconCls="icon-save" title="My Window">  

2.    Window Content   

3.  </div>  

 

1.  $('#win').window({   

2.      width:600,   

3.      height:400,   

4.      modal:true  

5.  });   

6.  $('#win').window('close');  // close window

boolean

boolean

true

boolean
