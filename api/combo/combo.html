扩展自 $.fn.validatebox.defaults 。 用 $.fn.combo.defaults 重写了 defaults 。

 

依赖

  • validatebox
  • panel

用法

1.  <input id="cc" value="001">  

2.  <select id="cc"></select>  

 

1.  $('#cc').combo({   

2.      required:true,   

3.      multiple:true  

4.  });
