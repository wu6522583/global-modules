扩展自 $.fn.window.defaults，用 $.fn.dialog.defaults 重写了 defaults。

 

依赖

  • window
  • linkbutton

用法

1.  <div id="dd" title="My Dialog" style="width:400px;height:200px;">  

2.      Dialog Content.   

3.  </div>  

 

1.  $('#dd').dialog({   

2.      modal:true  

3.  });  

特性

名称

类型

说明

string

boolean

false

minimizable

boolean

定义是否显示最小化按钮。

false

maximizable

boolean

定义是否显示最大化按钮。

false

resizable

boolean

定义对话框能否调整尺寸。

false

array

对话框的顶部工具栏，每个工具的选项都与 linkbutton 一样。

null

buttons

array

对话框的底部按钮，每个按钮的选项都与 linkbutton 一样。

null
