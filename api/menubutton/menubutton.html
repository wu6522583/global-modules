特性

其特性扩展自 linkbutton，下列是为 menubutton 增加的特性。

名称

类型

说明

默认值

plain

boolean

True 就显示一个简单效果。

true

menu

string

创建一个相应 menu 的选择器。

null

duration

number

当悬停在按钮上时，以毫秒为单位定义的，显示menu的持续时间。

100

名称

参数

说明
