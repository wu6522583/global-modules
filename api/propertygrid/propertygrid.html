用法

1.  <table id="pg"></table>  

 

1.  $('#pg').propertygrid({   

2.      url:'propertygrid_data.json',   

3.      showGroup:true  

4.  });  

特性

名称

类型

说明

默认值

showGroup

boolean

定义是否显示特性组。

false

groupField

string

定义组的字段名。

group

groupFormatter

function

定义如何格式化组的值。
